## 🚀 Overview

This application enables users to:

- Upload **multiple images** (`.jpg`, `.png`, etc.)
- Upload **multiple documents** (`.pdf`, `.docx`, `.txt`)
- Ask natural language questions
- Receive answers generated by **Gemini 1.5 Flash**
- Automatically extract and process only the relevant content

## 🖼 Features

- ✅ Supports uploading multiple images and documents
- ✅ Real-time chat with persistent history
- ✅ Dynamic context selection based on user input
- ✅ Clean and responsive UI built with Streamlit
- ✅ Secure API key handling using `secrets.toml`

## 📦 Installation

```bash
git clone https://github.com/your-username/gemini-flash-chatbot.git
cd gemini-flash-chatbot
pip install -r requirements.txt
```

## ⚙️ Configuration

Create a `.streamlit/secrets.toml` file and add your API key:

```toml
GEMINI_API_KEY = "your_google_generative_ai_key"
```

> ⚠️ **Important:** Do not commit this file. It contains sensitive credentials.

## 🧪 Usage

Run the app using Streamlit:

```bash
streamlit run app.py
```

Once running:

- Use the sidebar to upload your documents and images
- Ask a question like:
  > "What is the summary of this PDF?"  
  > "What do the uploaded images depict?"

- Receive real-time responses powered by **Gemini Flash**
